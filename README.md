# OCR Text Extractor - Streamlit App

Extract text from images using OCR and download as Word document.

## Features
- Upload multiple images (JPG, PNG, BMP)
- Extract text using EasyOCR
- Download extracted text as Word document
- Clean, professional UI

## Deployment
This app is deployed on Streamlit Cloud.

## Technologies
- Streamlit
- EasyOCR
- python-docx
- Pillow
